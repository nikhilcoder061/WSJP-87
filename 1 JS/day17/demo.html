What is HTML?
HTML stands for HyperText Markup Language. It is used to structure web pages.
What is CSS?
CSS stands for Cascading Style Sheets. It is used to style and layout web pages.
What is JavaScript?
JavaScript is a programming language used to make web pages interactive.
What is the difference between id and class in HTML?
An id is unique and used for a single element, while a class can be used for multiple elements.
What is responsive web design?
Responsive web design makes web pages look good on different devices by using flexible layouts.
